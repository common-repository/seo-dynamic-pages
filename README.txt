SEO Dynamic Pages allows a website owner to identify services their business offers as well as cities and towns. The plugin will dynamically create pages on your WordPress website with a combination of city and service without the need of physically creating each page.

== Installation ==

1. Move unzipped plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to SDP Settings to setup locations
4. Navigate to SDP Services and create new service
